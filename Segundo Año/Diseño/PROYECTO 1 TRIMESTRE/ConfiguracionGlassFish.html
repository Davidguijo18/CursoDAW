Configuración de GlassFish en Ubuntu

1. Configuración de Puertos

Por defecto, GlassFish usa el puerto 8080 para HTTP. Para cambiarlo, edita el archivo domain.xml en:

$GLASSFISH_HOME/glassfish/domains/domain1/config/domain.xml

Busca el bloque del conector HTTP y ajusta el puerto:

<network-listener port="9090" protocol="http-listener-1" .../>

2. Configuración de HTTPS

Para habilitar HTTPS, configura el conector SSL en el mismo archivo:

<network-listener port="8181" protocol="http-listener-2" .../>

3. Despliegue Automático

Para desplegar aplicaciones automáticamente, coloca los archivos WAR en:

$GLASSFISH_HOME/glassfish/domains/domain1/autodeploy/

4. Inicio Automático

Para que GlassFish se inicie automáticamente, crea un archivo de servicio:

sudo nano /etc/systemd/system/glassfish.service

Ejemplo de configuración:

        [Unit]
        Description=GlassFish Server
        After=network.target

        [Service]
        ExecStart=$GLASSFISH_HOME/bin/asadmin start-domain
        ExecStop=$GLASSFISH_HOME/bin/asadmin stop-domain
        User=glassfish
        Group=glassfish
        Restart=always

        [Install]
        WantedBy=multi-user.target

5. Pruebas

Accede a http://localhost:8080 (o el puerto configurado).
