Instalación de GlassFish en Windows

1. Requisitos Previos

Asegúrate de que tu sistema cumpla con los siguientes requisitos:

  • Windows 10 o superior.
  • Java Development Kit (JDK) 8 o superior instalado.
  • Conexión a Internet.

2. Instalación de Java

Verifica si Java está instalado en tu sistema. Abre una ventana de símbolo del sistema y escribe:

java -version

Si Java no está instalado, puedes descargarlo desde el sitio oficial de Oracle . Una vez descargado, sigue el proceso de instalación del JDK.

3. Descarga de GlassFish

Si aún no has descargado GlassFish, visita nuestra página de descargas y descarga la última versión de GlassFish. Una vez descargado el archivo, continúa con los siguientes pasos.

4. Instalación de GlassFish

Una vez descargado el archivo comprimido (por ejemplo, un archivo .zip), descomprímelo en la carpeta que desees, como C:\glassfish.

La estructura de carpetas resultante debería tener una carpeta llamada glassfish dentro de C:\glassfish con todos los archivos de instalación.

5. Configuración de Variables de Entorno

Configura las variables de entorno en tu sistema para que GlassFish funcione correctamente. Sigue estos pasos:

  • Haz clic derecho en "Este PC" y selecciona "Propiedades".
  • Haz clic en "Configuración avanzada del sistema".
  • En la ventana de propiedades del sistema, selecciona "Variables de entorno".
  • Bajo "Variables del sistema", haz clic en "Nueva..." y crea una variable llamada GLASSFISH_HOME con el valor de la ruta donde descomprimiste GlassFish (por ejemplo, C:\glassfish\glassfish).
  • Añade la carpeta bin de GlassFish al PATH. Crea o edita una variable de sistema llamada Path y añade la siguiente ruta: C:\glassfish\glassfish\bin.

6. Iniciar GlassFish

Para iniciar GlassFish, abre una ventana de símbolo del sistema y ejecuta el siguiente comando:

asadmin start-domain

GlassFish debería iniciar y podrás acceder a la consola de administración en http://localhost:4848 desde tu navegador.

7. Detener GlassFish

Para detener GlassFish, ejecuta el siguiente comando:

asadmin stop-domain
