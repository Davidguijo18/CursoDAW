Instalación de Apache Tomcat en Ubuntu

1. Requisitos Previos

Asegúrate de que tu sistema cumpla con los siguientes requisitos:

  • Ubuntu 20.04 o superior.
  • Java Development Kit (JDK) 11 o superior instalado.
  • Conexión a Internet.

2. Instalación de Java

Primero, verifica si Java ya está instalado:

java -version

Si no está instalado, puedes instalar OpenJDK 11 con los siguientes comandos:

sudo apt update sudo apt install openjdk-11-jdk

3. Descarga de Apache Tomcat

Si aún no has descargado Apache Tomcat, visita nuestra página de descargas descarga la última versión de Apache Tomcat. Una vez descargado el archivo, continúa con los siguientes pasos.

4. Instalación

Extrae el archivo descargado:

tar xvf apache-tomcat-9.0.73.tar.gz

Mueve la carpeta extraída a /opt:

sudo mv apache-tomcat-9.0.73 /opt/tomcat

5. Configuración

Configura las variables de entorno en el archivo ~/.bashrc:

export CATALINA_HOME=/opt/tomcat

Aplica los cambios:

source ~/.bashrc

6. Iniciar Apache Tomcat

Para iniciar Tomcat, ejecuta:

$CATALINA_HOME/bin/startup.sh

Para detenerlo:

$CATALINA_HOME/bin/shutdown.sh

7. Acceso a la Consola

Abre tu navegador y accede a http://localhost:8080 para verificar que Tomcat está funcionando.
