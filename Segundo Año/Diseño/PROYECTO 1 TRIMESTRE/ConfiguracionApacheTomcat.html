Configuración de Apache Tomcat en Ubuntu

1. Configuración de Usuarios y Roles

Para acceder a la consola de administración de Tomcat, edita el archivo tomcat-users.xml:

sudo nano /opt/tomcat/conf/tomcat-users.xml

Añade los siguientes roles y usuarios:

        <role rolename="manager-gui"/>
        <role rolename="admin-gui"/>
        <user username="admin" password="password" roles="manager-gui,admin-gui"/>

Guarda los cambios y reinicia Tomcat:

sudo /opt/tomcat/bin/shutdown.sh
sudo /opt/tomcat/bin/startup.sh

2. Configuración del Puerto

Por defecto, Tomcat usa el puerto 8080. Para cambiarlo, edita el archivo server.xml:

sudo nano /opt/tomcat/conf/server.xml

Busca esta línea y reemplaza el número de puerto:

<Connector port="8080" protocol="HTTP/1.1" .../>

Por ejemplo, para usar el puerto 9090, edita así:

<Connector port="9090" protocol="HTTP/1.1" .../>

3. Configuración del Firewall

Permite el puerto configurado en el firewall:

sudo ufw allow 8080/tcp

4. Configuración del Inicio Automático

Crea un archivo de servicio para que Tomcat se inicie automáticamente:

sudo nano /etc/systemd/system/tomcat.service

Añade el siguiente contenido:

        [Unit]
        Description=Apache Tomcat Web Server
        After=network.target

        [Service]
        Type=forking
        Environment=CATALINA_HOME=/opt/tomcat
        Environment=CATALINA_BASE=/opt/tomcat
        ExecStart=/opt/tomcat/bin/startup.sh
        ExecStop=/opt/tomcat/bin/shutdown.sh
        User=tomcat
        Group=tomcat
        Restart=always

        [Install]
        WantedBy=multi-user.target

Habilita el servicio:

sudo systemctl daemon-reload
sudo systemctl enable tomcat
sudo systemctl start tomcat

5. Prueba de Configuración

Abre tu navegador y accede a http://localhost:8080 (o el puerto que configuraste).
